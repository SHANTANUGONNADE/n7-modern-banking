# Image assets

Place your image files here. The app expects these filenames:

- Frame 114.png
- Frame 75.png
- Frame 90.png
- Frame 99.png
- frame 54.png
- Vector.png
- Vector-1.png
- Vector-2.png
- Vector-3.png
- iPhone 13 pro.png
- iPhone 13 Pro-1.png
- iPhone 13 Pro-2.png

**Important for Vercel:** File names are case-sensitive on Linux. Use the exact casing shown above (e.g. `frame 54.png` has a lowercase "f").
